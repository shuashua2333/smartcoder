# ⚡ SmartCoder 快速开始（5分钟）

## 🎯 准备工作

1. **在 VS Code 中打开 `smartcoder` 文件夹**（不是 `smart` 根目录！）
2. **打开终端**（`` Ctrl+` `` 或 视图 → 终端）

---

## 📝 步骤 1: 安装依赖

在终端中运行：

```bash
npm install
```

等待安装完成（1-2分钟）

---

## 📝 步骤 2: 编译扩展

在同一个终端中运行：

```bash
npm run compile
```

看到没有错误就 OK

---

## 📝 步骤 3: 配置 API Key

### 方法 A: VS Code 设置（推荐）

1. 按 `Ctrl + ,` 打开设置
2. 搜索 `smartcoder.apiKey`
3. 输入你的 DeepSeek API Key（`sk-xxxxx`）

### 方法 B: 编辑文件

打开 `index.html`，找到第 53 行，修改：

```javascript
const DEEPSEEK_KEY = "sk-你的实际API Key";
```

---

## 📝 步骤 4: 启动服务器

**打开新的终端窗口**（不要关闭之前的），运行：

```bash
cd server
npm install  # 如果还没安装
npm start
```

看到 `🚀 后端服务器启动: http://localhost:3000` 就成功了

**保持这个终端开着！**

---

## 📝 步骤 5: 启动扩展

在 VS Code 中按 `F5`

会打开一个新的 VS Code 窗口（Extension Development Host）

---

## 📝 步骤 6: 打开网页

1. 在文件管理器中找到 `smartcoder/index.html`
2. 双击用浏览器打开

---

## 📝 步骤 7: 测试

1. 在浏览器点击 **"🚀 在 VS Code 中编程"**
2. VS Code 应该自动打开并显示侧边栏
3. 编写代码
4. 点击 **"☁️ 提交到网页端"**
5. 切换回浏览器查看反馈

---

## ⚠️ 常见问题

### F5 没反应？
- 确保在 `smartcoder` 文件夹下
- 确保已运行 `npm install` 和 `npm run compile`

### 点击按钮没反应？
- 检查 `index.html` 中的 `EXTENSION_ID` 是否为 `smartcoder.smartcoder`
- 确保扩展开发主机窗口已打开

### 提交失败？
- 检查服务器是否在运行（`cd server && npm start`）
- 检查浏览器控制台（F12）是否有错误
